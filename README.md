
# Arduino Mini Projects

Apprentissage Arduino par mini projets progressifs.  
Approche pratique.  
Orientation système embarqué.

## Objectif

- Apprendre Arduino par la pratique.
- Comprendre le lien entre code et matériel.
- Structurer un projet embarqué proprement.
- Travailler sans interface graphique lourde.

## Organisation du dépôt



## Structure d’un projet

- README.md  
  - Objectif technique  
  - Principe de fonctionnement  
  - Schéma de câblage  
  - Matériel requis  
  - Instructions d’exécution  

- src/  
  - Code Arduino  

## Outils utilisés

- arduino-cli  
- Éditeur texte  
  - Neovim  
- Git  

Aucune dépendance à Arduino IDE.

## Matériel cible

- Arduino Nano  
- Composants standards  
  - LED  
  - Résistances  
  - Capteurs courants
